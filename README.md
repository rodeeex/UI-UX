# Notes App

Лёгкое веб-приложение для создания, редактирования, поиска и удаления заметок с тегами

## Автор

Пушкарев Антон, P4150

## Функциональность

- создание заметок
- редактирование заметок
- удаление заметок
- поиск заметок по тексту и тегам
- экспорт заметок в JSON-файл
- счётчик символов в тексте заметки
- сохранение данных в LocalStorage браузера

## Запуск

- Вариант 1. Открыть файл `index.html` в браузере
- Вариант 2. Запустить приложение по адресу https://rodeeex.github.io/UI-UX/

## Accessibility (A11Y)

Приложение соответствует базовым требованиям WCAG 2.1:

- семантическая вёрстка (`<header>`, `<main>`, `<dialog>`, `<ul>`)
- skip-link для перехода к основному содержимому
- `aria-атрибуты`
- `role="alert"` для ошибок валидации
- `role="search"` на панели поиска
- видимые индикаторы фокуса (`focus-visible`)
